# PanenKu
PanenKu: Marketplace pertanian modern untuk beli sayur &amp; buah segar langsung dari petani. Dibuat dengan Next.js &amp; MongoDB, aplikasi ini punya fitur keranjang belanja, manajemen stok otomatis, dashboard admin, dan label produk pintar. Tampilannya clean, premium, dan sangat simpel digunakan. Belanja hasil tani jadi jauh lebih gampang &amp; nyaman!
